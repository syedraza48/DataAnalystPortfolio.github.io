FG% Analysis Using SQL and Tableau

In this project, we joined two tables in PostgreSQL and displayed the results graphically using Tableau visualization tool.
The data was extracted from ESPN.com using Microsoft Excel. The excel files were then used to load data into our database where we created two tables; player_stats and team_stats. The query was stored in a view for easier access. We took our tabular results from the query and created a Tableau dashboard that shows each top 50 player's FG% next to their team's collective FG%.
This data is valid up to Jan 23, 2022. We will update the Tableau dashboard every Sunday night to reflect any new additions/subtractions to the top 50 pool.
Some things to note about the Tableau viz, not all NBA teams are mentioned here and there are multiple players listed from one team. For example, the Grizzlies have Ja Morant and Jaren Jackson in the top 50 pool. Also, the dotted line accross the bar graph represents the league average FG%.
What are some conclusions that we can draw from this dashboard? The best team in the Western Conference (PHO) and the best team in Eastern Conference (MIA) have players listed in this top 50 player study (Booker/Herro), however their FG% is lower than their team's FG% and lower than the league average FG%. The players with the best FG% (Sabonis/Jokic) are on teams that are having subpar seasons.
