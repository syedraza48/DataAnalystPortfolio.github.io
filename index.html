NBA Player/Team Analysis
Using SQL and Tableau

In this project, we joined two tables in PostgreSQL and displayed the results in a basic bar graph using Tableau visualization tool. The data was extracted from ESPN.com using Microsoft Excel. The excel files were then used to load data into our database where we created two tables- "player_stats" and "team_stats." The query was stored in a view for easier access. We then took our tabular results from the query and created a simple Tableau dashboard that shows each top 50 player's FG% next to their team's collective FG% with the league average FG% going accross the chart.
This data is valid up to Jan 23, 2022. We will update the Tableau dashboard every Sunday night to reflect all changes to the top 50 pool. This project will eventually attempt to predict who the MVP front runners are when we start adding more statistcs like W/L% and team rank.
